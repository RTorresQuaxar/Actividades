Programación
La programación es el proceso utilizado para idear y ordenar las acciones necesarias para realizar un proyecto.
SCRUM
Scrum es un proceso en el que se aplican de manera regular un conjunto de buenas prácticas para trabajar colaborativamente, en equipo, y obtener el mejor resultado posible
Accesibilidad Web
La accesibilidad web tiene como objetivo lograr que las páginas web sean utilizables por el máximo número de personas
